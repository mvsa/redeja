comandos git - 

git checkout -b "nome" - cria uma branch
git checkout master muda para a branch escolhida ( em exmplo a master)

git add -A - adiciona todas as alterações realizadas para o git
git commit -m "nome" - realiza o commit
git push origin "nome branch" - faz o push para o github
